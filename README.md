# Basic Computing Environment Setup

Resources for setting up computing environment.

## Windows

```powershell
powershell -File .\windows_setup.ps1
```

Install and activate Windows Subsystem for Linux (WSL).

```powershell
powershell -File .\wsl.ps1
```

## Linux

```bash
bash ./linux_setup.sh
```
